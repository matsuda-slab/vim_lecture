便利な応用技

テキストオブジェクト

  • 特定のモーションとコマンドの組み合わせで、便利な操作ができる
    • テキストオブジェクト 意味
    iw カーソル上の1単語選択
    ip カーソル上の段落を選択 (最後の改行含まない)
    ap カーソル上の段落を選択 (最後の改行含む)
    i" ""の中身を選択
    a" ""を選択
    i) ()の中身を選択
    a) ()を選択
    it タグの中身を選択 (html, xmlなど)
    at タグを選択 (html, xmlなど)
  • カッコのやつは, ( でも ) でもどちらでもよい

テキストオブジェクトの活用例

  • 1単語削除
    diw
  • 1単語削除して入力する
    ciw
  • "hogehoge"をヤンク
    ya"

レジスタの応用

  • 連番を一気に入力する
    • Try!
    1. 最下行で o で挿入モードに入り、a[0] と入力し、
      ノーマルモードに戻る
    2. yy で a[0]をヤンク
    3. 8p で 8行にわたってペースト
    4. 最初のa[0] の 0 にカーソルを持ってくる
    5. q a   (レジスタaに記録開始)
    6. v y   (0をヤンク)
    7. j
    8. v p   (一つ下にペースト)
    9. ctrl-a   (インクリメント)
    10. q   (記録終了)
    11. 7 @ a   (レジスタaの操作を7回繰り返す)

PacVim

  • パックマン風ゲーム
  • 遊び方
    $ ssh eagle
    $ cd ~matsuda/Share
    $ ./pacvim
  • ./pacvim 0 n   ←　ノーマルモード レベル0
  • ./pacvim 0 h   ← ハードモード レベル0
  • レベル : 0 ~ 9
  • 引数を指定しないと、自動でハードモード レベル0からのスタートになるので注意

  • 課題
    • PacVim を ノーマルモードでレベル5までクリアする
