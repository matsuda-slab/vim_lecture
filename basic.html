vimの基本操作

vimではモードを切り替えながら様々な操作を行う

  • ノーマルモード : 主にカーソルの移動, コピーペーストなど
  • インサートモード : 文字の入力
  • コマンドモード : コマンドの実行

  • ビジュアルモード : 範囲選択

  • (optional)Exモード

vimを使ってみよう

  1. 作業用のディレクトリを作ろう
    $ mkdir test

  2. vimを起動しよう
    • vim と入力して Enter を押すと、vimが起動する
    • vim {ファイル名} と入れて Enter を押すと、そのファイルをvimで開く
    $ vim hello.c

  3. 文字を入力してみよう
    • i を押すと、ノーマルモードからインサートモードに切り替わり、自由に文字入力ができる
      • キー 機能
      i カーソル位置に文字を入力
      a カーソルの後ろに文字を入力
      o カーソルの次の行に文字を入力
      s カーソルの文字を消して、その位置に文字を入力

  4. カーソルを動かそう
    • ESCキー を 2回 押すと、ノーマルモードに戻る
    • ノーマルモードで h j k l を押すと、それぞれカーソルが左 下 上 右に移動する
      • キー 機能
      h 左 に移動
      j 下 に移動
      k 上 に移動
      l 右 に移動

  5. ファイルを保存しよう
    • : でコマンドモードに切り替わる
    • コマンドモードで w と打ち、Enterをたたくと、保存される

  6. vimを終了しよう
    • コマンドモードで q と打ち、Enterをたたくと、vimを終了する
    • コマンドモードで wq と打ち、Enterをたたくと、保存と終了を一度に行う
